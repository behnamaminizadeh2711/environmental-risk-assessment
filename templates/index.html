Excel File Requirements

Your Excel file must have the following columns with the specified value ranges:

  • Risk Name: String (e.g., "A")
  • Probability (%): Number between 0 and 100 (for FMEA, e.g., 85)
  • Impact (Severity): Number between 1 and 10 (for FMEA, e.g., 8)
  • Detection: Number between 1 and 10 (for FMEA, e.g., 2)
  • Probability: Number between 1 and 5 (for Risk Matrix, e.g., 5)
  • Impact: Number between 1 and 5 (for Risk Matrix, e.g., 4)
  • Cause Likelihood: Number between 1 and 5 (for Bow-Tie, e.g., 5)
  • Consequence Severity: Number between 1 and 5 (for Bow-Tie, e.g., 4)
  • Barrier Effectiveness: Number between 1 and 5 (for Bow-Tie, e.g., 2)
  • Task Affected: String (e.g., "استخراج مواد اولیه")

Example Row: A, 85, 8, 2, 5, 4, 5, 4, 2, استخراج مواد اولیه
